RESTful API
GET:
Получить список ландшафтов из БД(id пользователя)
Загрузить ландшафт из БД(id)

POST:
Авторизация(логин, пароль)
Регистрация(логин, пароль, роль)
Сгенерировать ландшафт?(размер, сглаживание, цвет?)
Добавить ландшафт в БД(id пользователя, имя, размер, HeightsMap, TriPolMas, Color + параметры)

PUT:
Обновить ландшафт(HeightsMap, TriPolMas, Color)
Добавление пользователя модератором(id пользователя, id модератора)
Удаление пользователя модератором(id пользователя, id модератора)

PATCH:
Обновить параметры(width, height, range, smooth, mult, red, green, blue, size)

Delete:
Удалить пользователя из БД(id)
Удалить ландшафт из БД(id)
